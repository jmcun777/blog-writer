# China Briefing Sitewide Style Notes (Working Bank)

## Repeated strengths
- Utility-first titles that match user intent
- Guide-hub architecture (pillar + related nodes)
- Clear section progression: context -> implications -> action
- Dense but scan-friendly sectioning
- Frequent practical framing for foreign operators

## Reusable templates
- H2 style: short, intent-driven, practical
- Paragraph style: medium-length explanatory blocks with concrete qualifiers
- FAQ style: direct decision questions
- Sources style: authoritative + practical references

## Quality guardrails for our pipeline
- Situation first, operations second (unless explicitly ops SOP article)
- At least 2 tables for dense topics
- Keep headings short (3-8 words)
- Include boundary conditions and uncertainty notes
