# China Employee Termination in 2026: Pulse Exception-Ladder Brief for Foreign Employers

## Key takeaways
- Use an exception ladder to classify what blocks, what escalates, and what monitors.
- In China termination, route/evidence/payroll exceptions should default to top-tier blockers.
- Keep legal context first, then translate into practical intervention tiers.

## Plain-language legal context first
China is not an at-will system. Employer-initiated termination generally requires a lawful statutory route plus coherent evidence and procedural execution. Core anchors: Labor Contract Law, Decree No. 535 implementation rules, and SPC labor-dispute interpretation.

## Pulse exception ladder

### Tier 1 exceptions (hard block)
- Route rationale conflict
- Evidence chronology contradiction
- Settlement/payroll/tax mapping mismatch

**Action:** Immediate hold; legal + HR + payroll intervention before any progression.

### Tier 2 exceptions (expedite + monitor)
- Manager script drift
- Missing local city-validation memo

**Action:** Rapid correction window; promote to Tier 1 if unresolved near execution milestone.

### Tier 3 exceptions (closure gate)
- Incomplete closure evidence packet

**Action:** Keep case open; do not mark closed until archive completeness pass.

## Why this helps foreign employers
The ladder format gives non-local decision-makers a simple rule set: what absolutely stops the process versus what can be fixed in-flight. It reduces ambiguity during time-zone handoffs.

## What improved vs previous run
Compared with weighted control-index framing, this run improves operational clarity by converting weighted signals into explicit exception tiers with deterministic escalation pathways.

## Minimum documentation controls
- Canonical route memo
- Versioned chronology log
- Script acknowledgment record
- Settlement mapping sheet
- Payroll/tax reconciliation sheet
- City-level validation memo
- Closure evidence packet
