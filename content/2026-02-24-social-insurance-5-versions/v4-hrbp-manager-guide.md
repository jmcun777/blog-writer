# HRBP Manager Guide (2026): Handling Social Insurance for Foreign Employees in China

## Key takeaways
- HRBP teams need practical scripts and timing discipline.
- Managers should never improvise treaty or deduction explanations.
- Good communication before payroll day prevents most disputes.

## Manager-ready workflow
### Before onboarding
- Confirm role location and entity setup.
- Align with HR Ops on expected contribution handling.

### Before first payroll
- Use a standard script to explain deductions.
- Route exceptions to payroll/compliance instead of making verbal promises.

### After payroll
- Review employee questions in 24-48h.
- Document all resolved cases for knowledge base updates.

## HRBP document pack
- Employee profile summary
- Expected contribution sheet
- Treaty status note (if relevant)
- Communication template (email/chat)

## Escalation rules
- If treaty certificate absent: no exemption commitment.
- If local office has conflicting guidance: escalate and hold assumption.
- If employee challenge escalates: attach calculation + legal source references.

## Common mistakes for managers
- Overpromising exemption outcomes.
- Explaining deductions without payroll confirmation.
- Delayed response after payroll run.
