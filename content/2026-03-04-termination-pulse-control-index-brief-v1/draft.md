# China Employee Termination in 2026: Pulse Control-Index Brief for Foreign Employers

## Key takeaways
- Use a control index to score termination readiness before action.
- For foreign employers, route/evidence/payroll controls carry the highest weight.
- Keep legal framing upfront, then move to practical controls and thresholds.

## Plain-language legal context first
China is not an at-will market. Employer-initiated termination generally requires a lawful statutory route plus coherent evidence and process records. Core anchors: Labor Contract Law, Decree No. 535 implementation rules, and SPC labor-dispute interpretation.

## Pulse control index (weighted)

| Control area | Weight | Current status | Owner | Immediate action |
|---|---:|---|---|---|
| Route legality coherence | 25 | Amber | Legal + HR | Publish canonical route memo; freeze external steps until aligned |
| Evidence chronology integrity | 25 | Amber | HR Ops + Compliance | Reconcile version conflicts before progression |
| Settlement/payroll/tax mapping | 25 | Amber | Payroll + Legal | Pre-lock reconciliation and co-sign |
| Communication script discipline | 10 | Yellow | HRBP + Manager | Re-brief script and monitor next touchpoint |
| Local interpretation validation | 10 | Yellow | Local counsel + Compliance | Complete city memo before final action |
| Closure evidence completeness | 5 | Yellow | Ops + Compliance | Keep closure gate locked until packet complete |

**Index rule:** Any amber in a 25-weight control = no-go until resolved.

## Why this helps foreign employers
A weighted control index gives fast decision value for non-local leaders: which issue can block execution, which owner must act first, and which risks can be monitored.

## What improved vs previous run
Compared with risk-register format, this run improves prioritization by adding explicit weightings and a deterministic no-go rule for top-weight controls.

## Minimum documentation controls
- Canonical route memo
- Versioned chronology log
- Script acknowledgment record
- Settlement mapping sheet
- Payroll/tax reconciliation sheet
- City-level validation memo
- Closure evidence packet
