# China Employee Termination in 2026: Pulse Route-Map Brief for Foreign Employers

## Key takeaways
- Use a route-map format to avoid execution drift in China termination cases.
- Put legal route confirmation first, then navigate evidence, communication, payroll, and closure checkpoints.
- Route/evidence/payroll junctions remain the highest decision-risk intersections.

## Plain-language legal context first
China is not at-will. Employer-initiated termination generally requires a lawful statutory basis and coherent process evidence under the Labor Contract Law framework. Core anchors: Labor Contract Law, Decree No. 535 implementation rules, and SPC labor-dispute interpretation.

## Pulse route map

### Start: Route confirmation node
- Output: canonical route memo
- If unclear -> stop and re-validate legal basis

### Node 2: Evidence integrity node
- Output: versioned chronology log
- If contradictions -> hold progression

### Node 3: Communication node
- Output: script acknowledgment record
- If off-script risk -> re-brief before next touchpoint

### Node 4: Settlement/payroll node
- Output: settlement mapping + payroll/tax reconciliation
- If mismatch -> pre-lock no-go

### Node 5: Local validation node
- Output: city-level validation memo
- If missing -> block final action

### End node: Closure integrity
- Output: closure evidence packet
- If incomplete -> case remains open

## Why this helps foreign employers
A route map gives global teams a shared decision path with explicit checkpoints. It reduces handoff ambiguity across legal, HR, manager, and payroll functions.

## What improved vs previous run
Compared with watchlist format, this run improves procedural clarity by showing sequential decision flow and branch-based stop conditions.

## Minimum documentation controls
- Canonical route memo
- Versioned chronology log
- Script acknowledgment record
- Settlement mapping sheet
- Payroll/tax reconciliation sheet
- City-level validation memo
- Closure evidence packet
