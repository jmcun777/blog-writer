# China Employee Termination in 2026: Pulse Handoff-Protocol Brief for Foreign Employers

## Key takeaways
- Treat China termination as a handoff protocol, not isolated tasks.
- The highest-risk failures happen at legal→HR, HR→manager, and legal→payroll transitions.
- Keep legal context first, then enforce handoff check rules.

## Plain-language legal context first
China is not at-will. Employer-initiated termination generally requires a lawful statutory route plus coherent evidence and process consistency under the Labor Contract Law framework. Core anchors: Labor Contract Law, Decree No. 535 implementation rules, and SPC labor-dispute interpretation.

## Pulse handoff protocol

### Handoff 1: Legal -> HR (route lock)
- Required artifact: canonical route memo
- Pass condition: one consistent rationale
- Fail action: hold all employee-facing actions

### Handoff 2: HR -> Manager (communication lock)
- Required artifact: approved script acknowledgement
- Pass condition: script alignment confirmed
- Fail action: stop outreach; re-brief manager

### Handoff 3: HR/Compliance -> Legal (evidence lock)
- Required artifact: versioned chronology log
- Pass condition: no unresolved contradictions
- Fail action: pause progression

### Handoff 4: Legal -> Payroll (settlement lock)
- Required artifact: settlement mapping sheet
- Pass condition: term-to-pay code alignment complete
- Fail action: pre-lock no-go

### Handoff 5: Payroll -> Compliance (payment proof lock)
- Required artifact: payroll/tax reconciliation + payment proof
- Pass condition: reconciled and archived
- Fail action: keep closure blocked

### Handoff 6: Compliance -> Case Closure (archive lock)
- Required artifact: full closure packet
- Pass condition: closure checklist passed
- Fail action: case remains open

## Why this helps foreign employers
A handoff protocol makes execution ownership explicit across time zones. Instead of broad checklists, teams get deterministic transfer points with pass/fail criteria.

## What improved vs previous run
Compared with ops-kanban framing, this run improves accountability by defining mandatory handoff artifacts and lock conditions at each cross-functional transition.

## Minimum documentation controls
- Canonical route memo
- Versioned chronology log
- Script acknowledgment record
- Settlement mapping sheet
- Payroll/tax reconciliation sheet
- City-level validation memo
- Closure evidence packet
